Vocal-first. The voice is the lead organism. Generate expressive vocals first (using your fine-tuned Bark), then let the beat, bass, and harmony grow responsively around it. This naturally creates floating BPM and living groove.

## Recommended Hardware
- Windows 10 (64-bit)
- 16GB+ RAM (32GB better)

## Step-by-Step Installation (Limited Scope — Focused & Achievable)

### 1. Install the Launcher
Download **Pinokio** from pinokio.computer (single executable, very user-friendly).
Run it. This becomes your one-click control center for all local AI tools.

### 2. Install the Core Models via Pinokio
Inside Pinokio, search and install these (in this order):

1. **Ollama** or **LM Studio** — Local LLM for lyrics and prompt intelligence.
2. **Bark** — High-quality vocal generation. This is your primary vocal engine.
3. **AudioCraft / MusicGen** — For creating responsive accompaniment that follows the vocal.
4. **Ultimate Vocal Remover (UVR)** — Excellent GUI for stem separation on uploaded real audio.

Pinokio handles the heavy lifting. Just click Install on each.

### 3. Fine-Tune Bark on Your Voice (One-Time, High Impact)
This is the key to sovereignty and authenticity.
- Record several minutes of your singing in different styles/emotions.
- Use the fine-tuning tools available in the Bark section of Pinokio (or RVC if preferred).
- The model will now generate vocals that sound like *you* — your timbre, your phrasing, your frequency.

This step transforms generic AI output into bloodline-locked expression.

### 4. The Interface — Your Existing Suno.clone
Open the single-file `index.html` (the one in this repo) in any modern browser (Chrome/Edge recommended).

It already contains everything you need for creative control:
- Vocal-First guidance section
- Quantum parameters (Temperature, Nutrient Flow, Symbiosis, Fractal Coherence)
- Live multi-stem mixing and real-time stem export
- Mycelium network with lineage and grafting
- Persistent library
- Remix algorithms with distinct logic

### 5. The Vocal-First Workflow (Floating BPM)
1. Generate vocals in your fine-tuned Bark using text prompts or reference audio.
2. Export the vocal WAV.
3. Load the vocal into the browser app or note its timing/energy.
4. Use the procedural engine + quantum sliders to grow drums, bass, melody, and harmony that respond to the vocal's actual phrasing and energy.
5. The result is natural floating BPM — the beat breathes with the voice.

Export individual stems or the full mix whenever you want.

### 6. Optional but Recommended
- Use UVR when you want clean stems from any real uploaded tracks.
- Use the local LLM in Pinokio to generate better lyrics or structured prompts before sending them to Bark.
