# MYCELIUM v1.0 — Product Overview

**Sovereign Local Music Intelligence System**

## Executive Summary

MYCELIUM is a complete, production-ready, fully local music creation system designed for independent artists who demand sovereignty over their creative tools.

It solves the core problem of commercial AI music platforms: loss of control, data extraction, generic output, and recurring costs.

## The Problem

Current tools (Suno, Udio, etc.):
- Require ongoing subscriptions
- Train on contested public data
- Produce generic, safe results
- Offer limited voice control
- Lock users into cloud dependency

## The Solution

MYCELIUM gives the artist:
- Full local control
- Voice-locked generation (fine-tune on your own recordings)
- Vocal-first creative flow with natural floating BPM
- Perfect stem separation for generated material
- A beautiful, self-contained interface that works offline
- Complete lineage and variation tracking

## Target User

Independent musicians, especially those with a strong personal voice and story (Rust Belt, trauma-to-signal, fatherhood, legacy themes) who want technology that amplifies rather than replaces their signal.

## Core Innovation

**Vocal-First + Adaptive Mycelium**

Instead of generating full songs in one shot on a rigid grid, MYCELIUM lets the artist generate the vocal first (with their own fine-tuned voice), then grows the accompaniment responsively around the vocal's actual timing and energy. This produces music that breathes.

## Technical Stack

**Interface Layer**
- Single-file HTML/JS/CSS (no build step, no dependencies)
- Real-time Web Audio API procedural synthesis
- Interactive Canvas-based Mycelium network graph
- Full offline stem rendering and export

**Local AI Layer**
- Bark (fine-tunable) for expressive vocal generation
- MusicGen for adaptive, vocal-conditioned accompaniment
- UVR (Ultimate Vocal Remover) for stem separation on real audio
- Local LLM for prompt/lyric assistance

**Data Philosophy**
- 100% user data only after initial setup
- Fine-tuning supported on personal recordings
- No telemetry, no cloud sync required

## Version 1.0 Scope

This release focuses on delivering a complete, usable, sovereign workflow:
- Strong vocal generation foundation
- Excellent stem-level control
- Beautiful creative interface
- Clear path to voice-locked personalization

Future versions will deepen the adaptive accompaniment intelligence and add more sovereign training loops.
